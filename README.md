# 阿瓦隆 — 历史风云

> 天下大势，分久必合，合久必分。谁是忠臣，谁是奸佞？

单文件阿瓦隆（The Resistance: Avalon）桌游 Web 版，以中国历史人物为角色，接入 Claude API 实现 AI 自主推理与对局。

## 特色

- **单文件部署**：一个 `avalon.html` 即可运行，无需后端、无需构建
- **AI 驱动**：接入 Claude API（claude-opus-4-20250514），AI 玩家具备完整的策略推理能力
- **历史人物角色**：张居正、秦始皇、武则天、苏轼、霍去病、李清照、朱棣、诸葛亮、李世民、曹操——每人拥有独特性格和说话风格
- **古风 UI**：暗色调中式美学，Noto Serif SC 字体，金/红/蓝配色
- **完整规则实现**：5–10 人局，梅林、派西维尔、莫德雷德、刺客、莫甘娜、奥伯伦全角色支持
- **人机混战**：支持任意数量真人与 AI 混合对局
- **AI 思考可视化**：点击 [思考] 展开 AI 的内心推理过程

## 游戏规则

阿瓦隆是一个社交推理游戏，玩家分为正义与邪恶两个阵营：

- **正义阵营**：梅林（知晓邪恶）、派西维尔（知晓梅林/莫甘娜）、忠臣
- **邪恶阵营**：莫德雷德（梅林不可见）、刺客（可暗杀梅林）、莫甘娜（伪装梅林）、奥伯伦（孤狼）

每轮由队长提议出征队伍 → 全体投票 → 通过则执行任务（秘密出牌）。正义方先赢三轮任务则胜，但刺客有最后机会刺杀梅林翻盘。连续 5 次否决提案则邪恶直接获胜。

## AI 策略系统

AI 玩家不是简单的随机决策，而是具备深度策略推理：

- **邪恶方**：伪装好人发言、战略性出牌（前期建立信任 vs 关键局出击）、隐式信号配合同伴
- **梅林**：故意制造投票噪音保护身份、模糊暗示而非直接指控
- **派西维尔**：分析梅林/莫甘娜行为差异、跟随真梅林判断分散刺客注意
- **嫌疑人矩阵**：自动交叉比对任务结果，追踪每位玩家的出征成功/失败记录
- **投票模式分析**：计算玩家间投票一致率，识别可疑同盟关系

## 使用方法

1. 浏览器打开上方链接或本地 `avalon.html`
2. 输入 Anthropic API Key（或留空使用内置认证）
3. 选择人数（5–10 人），为每个座位设置角色名和类型（AI / 真人）
4. 点击「开始远征」

## 技术细节

- 纯前端，HTML + CSS + JavaScript 单文件
- 直接调用 Anthropic Messages API（需 `anthropic-dangerous-direct-browser-access` header）
- 内置请求限流重试（429）和超时处理（45s）
- AI 无响应时自动使用角色特色备用台词
- JSON 解析容错：支持截断 JSON 修复
